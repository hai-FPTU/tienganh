🔑 Kết nối AI — cài 1 lần cho máy này

Chấm điểm, biểu đồ và nhận xét cơ bản luôn chạy được, kể cả không có mạng. Dán API key Gemini nếu muốn có thêm phần AI giải thích từng lỗi.

📌 Cách lấy API key Gemini miễn phí (2 phút)
  1. Mở aistudio.google.com/apikey trên trình duyệt, đăng nhập tài khoản Google.
  2. Bấm Create API key (Tạo khóa API) → chọn hoặc tạo một project.
  3. Bấm biểu tượng Copy để sao chép chuỗi key (bắt đầu bằng AIza…).
  4. Quay lại trang này, dán vào ô API key → bấm 💾 Lưu API key → bấm Kiểm tra kết nối.

Gói miễn phí hiện chỉ áp dụng cho các mô hình Flash và Flash-Lite (các bản Pro đã chuyển sang trả phí từ 01/4/2026), không cần thẻ tín dụng, khoảng 1.500 lượt/ngày — thừa đủ cho một lớp. Nếu báo lỗi 429 nghĩa là gọi quá nhanh, chờ khoảng 1 phút rồi thử lại.

Key là của riêng thầy cô, đừng đưa cho học sinh. Key chỉ nằm trong trình duyệt máy này; nếu là máy dùng chung, bấm Xóa key sau khi dùng.

Lưu ý riêng tư: ở gói miễn phí, Google có thể dùng nội dung gửi lên để cải thiện sản phẩm. Nên cho học sinh ghi tên gọi thân mật thay vì họ tên đầy đủ.
