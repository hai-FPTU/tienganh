    Nhờ AI giảng lại chỗ sai

    AI sẽ giải thích bằng tiếng Việt: vì sao sai, quy tắc, ví dụ, và kế hoạch luyện tuần tới. Phần này cần mạng và cần khóa API Gemini của thầy cô. Khóa chỉ nằm trong bộ nhớ của trình duyệt lúc này — đóng trang là mất, file không lưu khóa nào cả.

    Lấy khóa API Gemini ở đâu? (2 phút)
    1. Mở aistudio.google.com trên trình duyệt, đăng nhập tài khoản Google.
    2. Ở cột trái chọn API keys (hoặc nút Get API key).
    3. Bấm Create API key, chọn một dự án Google Cloud (không cần thẻ thanh toán cho gói miễn phí).
    4. Bấm biểu tượng sao chép. Khóa có dạng AIzaSy….
    5. Dán vào ô Khóa API Gemini ở trên rồi bấm Tải danh sách model để kiểm tra khóa chạy được.

    Nếu báo lỗi 403 “restricted key”: Google yêu cầu khóa mới phải có giới hạn. Trong AI Studio / Google Cloud, mở khóa đó và đặt API restrictions → Generative Language API. Đừng dùng giới hạn theo tên miền (HTTP referrer) nếu thầy cô mở file này trực tiếp từ máy, vì file cục bộ không gửi tên miền.

    Lưu ý an toàn: khóa là của riêng thầy cô. Đừng dán khóa vào máy của học sinh; cách dùng gọn nhất là học sinh làm bài và chấm offline, thầy cô mở phần AI trên máy mình.

    Dành cho thầy cô — tự kiểm tra máy chấm & đáp án

    Hai nút dưới đây tự điền bài rồi tự chấm, để kiểm chứng phần chấm điểm hoạt động đúng.
